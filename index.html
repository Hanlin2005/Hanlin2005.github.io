Pointers to Interesting Reads

The internet houses an ocean of resources of varying quality. Once in a while, stashed in a forgotten corner of the internet, I can find a shining gem that makes me appreciate the fact that I was born in this era. In fact, these blogs gave me the inspiration to design this website. I'd like to share some of these resources just in case they might be inspiring to you!

  • Birds and Frogs - Written by Freeman Dyson, an interesting read for aspriring mathematicians (and physicists)
  • Henry Greenside's Physics Challenges - A collection of physics challenges!
  • Physics Resources from Zero Hero - You can already tell I wanted to be a physicist.
  • Online Deep Learning Textbook - Great resource to learn deep learning
  • Some Algorithm Puzzles - One of my favorite personal websites out there
  • 4 gravitons - Physics blog!
  • Computational Fluid Dynamics in Python - A great resource for learning CFD
  • Spinning Up in Deep RL - Resource for learning deep reinforcement learning
